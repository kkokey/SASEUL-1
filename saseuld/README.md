# saseuld

## Installation

Clone the repository.

```sh
git clone git@github.com:Artifriends-inc/saseul-origin.git
```

Change working directory.

```sh
cd saseul-origin/saseuld
```

Install all the dependencies using composer.

```sh
composer install
```

## Run tests

```bash
composer test
```

## Run fixer

```bash
composer fixer
```
